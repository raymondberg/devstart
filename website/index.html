DevStart

About the Project

  • A custom experience to move each participant("student") from their current state to the next level. (technophobe -> game player, dabbler -> beginning programmer, geek -> ubergeek).
  • Each session starts with 1 microtalk - a 10-15 minute discussion to explain new fundamental concepts (basic commands, loops/blocks, object oriented design, language basics, technologies[web/scripts/apps])
  • Resources will be geared toward the 10-17 age range, but since this is a first offering that age group is flexible. Parents are welcome to participate with their children for none/part/all of the activities.
  • Remainder of time will be divided between participants for:
    • General coaching - talk out harder problems to schoolwork or personal work
    • Guided "play throughs" - Self-Guided Resources offer basic games and tools to start programming but can leave students stuck without guidance
    • Custom Projects - Provide and guide solvable, real-wold problems custom-designed for student skill level.
    • Team development work - Partnering participants of similar skill levels to start working together to solve programming problems

*Note: Many public schools offer curriculum that is far more targeted and specialized for classrooms and age ranges. This is not intended as a substitute or replacement. Be sure to look for Computer Science, Math and Programming options in your local school system!!

How / Where / When

  • Buy-in cost: $32. Coach(es) are all-volunteer. Money will be used to buy classroom materials or donated to a STEM education-related non-profit.
  • Students must come to 4 of 6 sessions to remain in course.
  • Additional requirements: Laptops** to bring to sessions and for completing personal work. Parents/Guardians with administrative rights to install software if needed.

**Note: Reach out to coach if this is the only thing holding you back! The long term goal is to provide affordable laptops as part of charity-sponsored course offerings, but we can see what to try in the meantime.

Initial offering is internal only, but grab me on Twitter (@raymondberg) if you are interested!

*Note: Possible extension or second offering based on demand/interest.

Self-Guided Resources

Lots of resources exist to learn how to program, but here are the best free ones money can't buy!

Websites

Code.org ("one hour of code")

Hands down the cleanest first exposure to programming. Walk through a guided introduction to computing basics with fun games, video explanations and more.

Scratch

The original animation scratchpad. Develop cool animations using drag/drop code.

Bot Logic

Quick and simple game for early learners. Start with drag/drop and then start writing your own code by hand.

CodeCombat

Crawl dungeons and fight monsters while learning language concepts. This one gets tricky fast! (Tip: Don't get stuck: save gems unless you need to spend!)

Made With Code (by Google)

Games, projects and more ways to stretch your brain with drag/drop code.
