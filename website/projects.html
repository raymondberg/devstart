Devstart/Projects

Tackle these problems in order for best results!

Easy!

The first line of your code is this:

notes = ["do","re","mi","fa","so","la","ti","do"]

The way your program should work:

  • Say the notes of the scale up and down
  • You can do it once the manual way, but then do it with a loop or two :)

Example

    do
    re
    me
    fa
    so
    la
    ti
    do
    do
    ti
    la
    so
    fa
    me
    re
    do

Clue: What happens when you type `len(notes)` in the Python Console? What about `range(1,10)`? What about `range(10,1)`? What about `range(10,1,-1)`?

Medium!

The first line of your code is this:

dinosaurs = ["T-Rex", "Pterodactyl", "Triceratops", "Barney"]

The way your program should work:

  • Ask the users in what year they think the dinosaurs lived (BCE)
  • Then say "If that's true then..." and list off when the dinosaurs lived
  • Each dinosaur should live one year after the previous, starting with the year +1 (or minus 1 because it's BCE!)

Example

    In what year do you think the dinosaurs lived? (user enters 60000)
    If that's true then...
    T-Rex lived in the year 59999 BCE
    Pterodactyl lived in the year 59998 BCE
    Triceratops lived in the year 59997 BCE
    Barney lived in the year 59996 BCE

Clue: It's tricky to mix strings and numbers. What happens when you type `str(9)` in the Python Console? What about `"The number %d." % 9` ?

Hard!

The first line of your code is this: 

things = ["Armadillos", "Muppets", "Spooky Houses", "Black Labs", "Bananas"]

The way your program should work:

You are going to ask the user about their likes/dislikes

  • Tell the user you would like to ask them if they like a few things or not.
  • Tell the user to specify their answer between 1 and 5
    • If the user types in 5, they REAAAAALLY love the thing
    • If the user types in 1, they REEEALLY don't like the thing
  • For each item in the list of things, ask the user how they feel
  • After the user enters how they feel, say how they feel back to them. This should change based on what the user says.
  • When they are all done, thank them for answering their questions. Maybe come up with something funny to say. This doesn't have to change. Just put some plain old text in there.

Example

    I want to ask you how you feel  some things. Answer between 1 and 5.
    How do you feel about Armadillos? (the user enters 1)
    Oh, you REAAAAAALLY don't like Armadillos. Got it!
    How do you feel about Muppets? (the user enters 4)
    Oh, you really like Muppets. Got it!

    ...
    Hmm, from what you told me you seem to think Martians are going to take over the earth. Very interesting. Thanks for talking to me.

Super Hard!!

The first line of your code is this:

 items = []

The way your program should work:

You are going to let the users create their own lists as they go, basically pretending to be python while a user types.

  • Start by telling the users what their list looks like.
  • Then, ask what the user to enter a command.
  • After each command is complete you should print what the list looks like again
  • The user should be able to do 3 things: stack, unstack, quit
    • When the user chooses list, they should also type in an item they want to list. This item will get added from the array.
    • When the user chooses unstack, the last item on the list should get removed
    • When the user chooses list, tell the user to "Have a nice day" and stop asking for more input

Example

    Greetings! I have made a list for you.
    This is what your list looks like: []
    What should I do? (user enters stack 3)
    This is what your list looks like: ["3"]
    What should I do? (user enters stack banana)
    This is what your list looks like: ["3","banana"]
    What should I do? (user enters stack lake)
    This is what your list looks like: ["3","banana","lake"]
    What should I do? (user enters unstack)
    This is what your list looks like: ["3","banana"]
    What should I do? (user enters quit)
    Have a nice day!

Clue: What happens when you type `"first second".split(" ")` in the Python Console?

Crazy Hard

Build on the code you made in the last project

  • Add a 4th command: laserblast
  • laserblast can remove a single item from anywhere in the list, regardless of its position
  • When a user specifies a laserblast they should also type in the item they want to remove
  • When a user enters another command tell them that you don't understand the command and ask again for command

Example

    This is what your list looks like: []
    What should I do? (user enters stack 3)
    This is what your list looks like: ["3"]
    What should I do? (user enters unstork)

    This is what your list looks like: ["3","banana"]
    What should I do? (user enters stack lake)
    This is what your list looks like: ["3","banana","lake"]
